# Doctrine Contracts

A set of abstractions for doctrine.

### Installation

```bash
$ composer require siganushka/doctrine-contracts
```

### Interfaces & Traits

```
Siganushka\Contracts\Doctrine\ResourceInterface
Siganushka\Contracts\Doctrine\ResourceTrait
```
